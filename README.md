# Dementia Support App

A dedicated tablet application designed to provide gentle, automated orientation and support for individuals with dementia. 

## Features
- **Passive Orientation (Wake Screen):** Displays a highly-readable time, date, and comforting messages to reduce anxiety and confusion.
- **Natural AI Voice Integration:** Incorporates high-quality, soothing neural text-to-speech to read messages and reduce reliance on robotic, jarring standard TTS.
- **Cognitive Load Management:** A heavily simplified interface prioritizing contrast, readability, and single-action functionality.
- **Caregiver Controls:** Dedicated panels for family and caregivers to handle setup and alarms remotely.

## Getting Started

To run the application locally on your machine:

1. Install the required dependencies:
   ```bash
   npm install
   ```
2. Start the Vite development server:
   ```bash
   npm run dev
   ```

## Tech Stack
- React 18 + Vite
- Tailwind CSS
- Radix UI Primitives
- Framer Motion
